    Flash Bulb for jQuery

    Flash Bulb will animate a set of elements onto of off of the page in a sequential order.

    Basic Implementation

    Basic structure of elements. 
    <div class="flashBulb">
  <div>subelement</div>
  <div>subelement</div>
</div>
    This will animate the elements onto the page when the page loads. 
    $(document).ready(function() {
  $('.flashBulb div').flashBulb();
});

    // sample code for example below.
$(document).ready(function() {
    $elements = $('.demo-container div');
    $elements.hide();
    $('#show').click(function() {
        $elements.flashBulb({
            method: 'show',
            fadeSpeed: 600,
            interval: 200
        });
    });
    $('#kill').click(function() {
        $elements.flashBulb('kill');
    });
    $('#hide').click(function() {
        $elements.flashBulb({
            method: 'hide',
            fadeSpeed: 600,
            interval: 200
        });
    });
});

    Options

    Flash Bulb accepts single string arguments of 'show', 'hide', or 'kill'. It also accepts options passed in within an object, as well as a callback function which will be called as soon as the elements are completely finished animating. 
    
method: 'show', // 'show', 'hide', 'kill'
initialFlash: 1000, // speed of the inital fade animation
fadeSpeed: 1000, // speed that the flash fades out
slideSpeed: 'slow', // speed that surrounding elements are push out of the way. 'slow', 'fast', or integer number
interval: 400 // time between element animations

    Interrupting the current animation

    $('.flash').flashBulb('kill'); // will interrupt the animation sequence
